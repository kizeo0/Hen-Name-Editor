# Hen-Name-Editor
Este programa permite cambiar el nombre del icono "Enable HEN" en tu PS3 de forma automática, generando un archivo instalador (.pkg) que puedes llevar en un USB.

¿ CUAL USAR ?


AUTOHEN=Instalado desde el Videohost/cualquier web con autohen


HEN OFICIAL=Instalado desde PS3XPLOIT

Ambos testeados en 4.92.
